Manager plugins. Supported platforms: Kubernetes

Usage:
  tkeel plugin [flags]
  tkeel plugin [command]

Examples:

# Manager plugins. in Kubernetes mode
tkeel plugin list -k
tkeel plugin delete -k
tkeel plugin register -k


Available Commands:
  delete      Register plugins. Supported platforms: Kubernetes
  list        Show the health status of TKeel plugins. Supported platforms: Kubernetes
  register    Register plugins. Supported platforms: Kubernetes

Flags:
  -h, --help         Print this help message
  -k, --kubernetes   List tenant's enabled plugins in a Kubernetes cluster (default true)

Global Flags:
      --log-as-json   Log output in JSON format

Use "tkeel plugin [command] --help" for more information about a command.
